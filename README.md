# Hospital-Emergency-Room-Dashboard


Project overview:

This project focuses on analyzing emergency room patient data to monitor hospital performance , patient flow , and service efficiency. This dashboard helps identify waiting time trends , patient demographics, admission patterns , and department referrals.

🎯 Objective

To analyze emergency room data and provide insights that help improve patient management, reduce waiting time, and enhance hospital service quality.

🗂 Dataset Description

The dataset contains patient emergency visit records including:

• Patient count
• Waiting time
• Admission status
• Patient age group
• Department referrals
• Gender distribution
• Patient satisfaction score

⚙ Tools & Techniques Used

• Microsoft Excel
• Power Query
• Power pivot
• Pivot Tables
• Data Cleaning
• Data Visualization
• Interactive Dashboard Design

🔄 Project Process

✔ Cleaned and prepared raw healthcare data
✔ Created pivot tables for analysis
✔ Designed interactive dashboard with slicers
✔ Developed KPI metrics to monitor hospital performance

📊 Dashboard Features

• Total patient count tracking
• Average waiting time monitoring
• Patient satisfaction analysis
• Admission vs Non-Admission breakdown
• Patient age group analysis
• Department referral analysis
• Gender distribution analysis

🔍 Key Insights

• Nearly equal ratio of admitted vs non-admitted patients
• Majority of emergency patients fall under younger age groups
• General Practice receives highest patient referrals
• Waiting time directly impacts satisfaction score

💡 Business Recommendations

• Optimize staffing during high patient volume periods
• Improve waiting time management strategies
• Enhance support for high referral departments
• Focus on improving patient experience
